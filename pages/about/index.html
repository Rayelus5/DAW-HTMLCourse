Misión del portal web

Este portal web nace con el objetivo de acercar el trabajo del alumnado con el profesorado del módulo Lenguajes de Marcas impartido en el Ciclo Formativo de Grado Superior de Desarrollo de Aplicaciones Web (DAW).

El código de esta web tanto como el de las actividades es OpenSource, es decir, público y de código abierto. Puede ser compartida con fines educativos.
